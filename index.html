03 — What I do best

Patterns I reach for.

Reliable event delivery

Transactional outbox + dead-letter queue with replay, so a DB write and a published event never diverge.

Idempotent payments

Stripe/Moneris webhooks with signature validation + idempotency keys — no double charges, even on retries.

Caching & scale

Redis cache-aside with event-driven invalidation on hot read paths — cutting redundant queries without stale data.

Multi-tenant isolation

PostgreSQL Row-Level Security with per-request tenant context — defense in depth, not trust-the-developer.

Hardened auth

JWT with refresh-token rotation, HttpOnly cookies, TOTP 2FA, and RBAC guards across customer-facing APIs.

Real-time & async

Socket.IO scaled with a Redis adapter, plus RabbitMQ/BullMQ workers for background jobs and notifications.
